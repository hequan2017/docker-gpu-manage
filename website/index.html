快速开始

3 分钟快速部署

支持多种部署方式，选择最适合您的方式快速启动

Docker 快速部署 
# 1. 克隆项目
git clone https://github.com/he7555/docker-gpu-manage.git
cd docker-gpu-manage

# 2. 使用 Docker Compose 启动
docker-compose up -d

# 3. 访问管理界面
open http://localhost:8888
Kubernetes 部署 
# 1. 添加 Helm 仓库
helm repo add gpu-manage https://charts.gpu-manage.io

# 2. 安装到 Kubernetes 集群
helm install gpu-manage gpu-manage/gpu-manage \
  --namespace gpu-manage \
  --create-namespace

# 3. 等待 Pod 就绪
kubectl wait --for=condition=ready pod \
  -l app=gpu-manage \
  -n gpu-manage \
  --timeout=300s
本地开发环境 
# 1. 后端启动
cd server
go mod download
go run main.go

# 2. 前端启动 (新终端)
cd web
npm install
npm run dev

# 3. 访问开发环境
# 前端: http://localhost:3000
# 后端: http://localhost:8888

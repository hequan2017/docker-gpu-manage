# 戴尔资产管理

## 概述

戴尔资产管理功能用于管理 Dell 服务器硬件资产的生命周期，包括硬件信息记录、物理位置追踪、保修期管理和状态监控。通过统一的资产管理平台，实现服务器资产的规范化管理和运维支持。

## 功能特性

### 核心功能

- ✅ **硬件信息管理**：记录 CPU、内存、磁盘、网卡等硬件配置
- ✅ **服务标签追踪**：使用 Dell Service Tag 进行唯一标识
- ✅ **物理位置管理**：记录机柜和机架位置（U位）
- ✅ **保修期管理**：追踪购买日期和保修到期日
- ✅ **状态监控**：实时监控服务器电源和运行状态
- ✅ **资产统计**：提供多维度的资产统计分析

## 使用指南

### 添加资产

1. 进入【戴尔资产】管理页面
2. 点击"新建资产"按钮
3. 填写资产信息：

#### 基本信息区
- **主机名**：服务器主机名
- **服务标签**：Dell Service Tag（必填，唯一标识）
- **资产编号**：内部资产管理编号
- **型号**：服务器型号（如 PowerEdge R740）
- **序列号**：出厂序列号

#### 硬件配置区
- **CPU 型号**：如 Intel Xeon Gold 6248R
- **CPU 核心数**：物理核心数
- **CPU 线程数**：逻辑线程数
- **内存容量**：单位 GB
- **磁盘信息**：磁盘类型、容量、RAID配置
- **网卡信息**：网卡型号、端口数量

#### 网络配置区
- **IP 地址**：管理网口 IP
- **MAC 地址**：管理网口 MAC

#### 物理位置区
- **机柜位置**：机柜编号（如 A01、B02）
- **机架位置**：U 位高度（如 U12-U15）

#### 保修信息区
- **购买日期**：设备购买日期
- **保修到期日**：保修期结束日期

#### 管理信息区
- **操作系统**：安装的操作系统
- **所属部门**：资产归属部门
- **负责人**：资产责任人
- **状态**：在线/离线/维护中
- **备注**：其他说明信息

4. 点击"确定"添加资产

### 编辑资产

1. 在资产列表中找到要编辑的资产
2. 点击"编辑"按钮
3. 修改资产信息
4. 点击"确定"保存更改

### 删除资产

**单个删除：**
1. 在资产列表中找到要删除的资产
2. 点击"删除"按钮
3. 确认删除操作

**批量删除：**
1. 勾选多个资产
2. 点击"批量删除"按钮
3. 确认删除操作

### 查看资产统计

系统提供多维度的资产统计信息：

- **资产总数**：记录的资产总数量
- **在线数量**：当前在线的资产数量
- **离线数量**：当前离线的资产数量
- **维护中数量**：维护状态的资产数量
- **保修到期提醒**：即将过保的资产列表

## Service Tag 说明

### 什么是 Service Tag？

Dell Service Tag 是戴尔产品的唯一标识符，由 7 位字符组成，用于：
- 唯一标识 Dell 服务器
- 查询保修状态和技术支持
- 获取驱动程序和更新
- 追踪设备生命周期

### 查找 Service Tag

**物理标签位置：**
- 机箱正面或背面
- 机箱内部标签
- 原始包装箱

**系统内查询：**
```bash
# Linux 系统
sudo dmidecode -s system-serial-number

# Windows 系统
wmic bios get serialnumber

# 通过 iDRAC
racadm getsysteminfo
```

### Service Tag 用途

1. **保修查询**：访问 Dell 官网查询保修状态
2. **技术支持**：联系 Dell 技术支持时提供
3. **驱动更新**：自动识别适用的驱动程序
4. **资产追踪**：作为资产唯一标识

## 物理位置管理

### 机柜编号规范

建议使用统一的机柜编号规则：

```
格式：[区域]-[行]-[列]
示例：A-01-05（A区01排05列）
```

### 机架 U 位规范

U 位是机架的标准高度单位（1U = 1.75 英寸 = 4.445 cm）：

```
格式：起始U-结束U
示例：U12-U15（占用4U高度）
```

**记录规范：**
- 从下往上编号（U1 在底部）
- 记录设备占用的起始和结束 U 位
- 单台设备：记录其所在 U 位
- 多台设备：记录占用的 U 位范围

## 保修管理

### 保修状态查询

1. 访问 Dell 官网保修查询页面
2. 输入 Service Tag
3. 查看保修状态和到期日
4. 更新系统中的保修信息

### 保修到期提醒

系统会自动检测即将到期的资产：

- **即将到期**：30天内到期
- **已到期**：已过保修期
- **在保**：保修期内

建议设置定期提醒，提前规划设备更新或续保。

### 状态类型

|------|------|------|

| offline | 离线停机 | ⚫ |

1. 在资产列表中找到要操作的资产
2. 点击状态下拉菜单
3. 选择新的状态
4. 确认操作

## 资产导入导出

### 批量导入

支持通过 Excel/CSV 批量导入资产信息：

1. 下载导入模板
2. 按模板格式填写资产信息
3. 上传填写好的文件
4. 系统自动解析并导入

**导入字段映射：**
- 主机名 → hostName
- 服务标签 → serviceTag
- 型号 → model
- IP地址 → ipAddress
- 机柜位置 → cabinet
- ...

### 导出资产

1. 筛选要导出的资产
2. 点击"导出"按钮
3. 选择导出格式（Excel/CSV）
4. 下载导出文件

## 常见问题

### Q1: 如何获取 Dell 服务器的 Service Tag？

A: 有以下几种方式：
1. 查看机箱上的物理标签（正面或背面）
2. 登录 iDRAC 查看系统信息
3. 使用命令查询：
   ```bash
   # Linux
   sudo dmidecode -s system-serial-number

   # Windows PowerShell
   Get-WmiObject Win32_BIOS | Select-Object SerialNumber
   ```

### Q2: 机架 U 位如何计算？

A:
- 1U = 4.445 cm（1.75 英寸）
- 标准机柜高度为 42U
- U 位从下往上编号，底部为 U1
- 记录时应标注设备占用的 U 位范围

### Q3: 保修到期后如何处理？

A:
1. 联系 Dell 销售咨询续保方案
2. 评估设备性能，考虑更换新设备
3. 更新系统中的资产状态
4. 调整设备的使用策略

### Q4: 如何查询 Dell 服务器保修状态？

A:
1. 访问 Dell 官网保修查询页面
2. 输入 Service Tag
3. 查看保修类型和到期日期
4. 更新到资产管理系统中

### Q5: 资产信息可以批量修改吗？

A:
- 支持批量删除操作
- 批量修改功能可通过导入导出实现
- 导出数据 → Excel 批量修改 → 重新导入
